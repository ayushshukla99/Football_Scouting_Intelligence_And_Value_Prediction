# Football Transfer Market Value Prediction & Scouting Intelligence Dashboard

## 📌 Project Overview
This project builds an end-to-end data analytics and machine learning system to predict football player market value and identify undervalued players for scouting purposes.

## 🎯 Business Problem
Football clubs often rely on subjective scouting and incomplete data when evaluating player transfers. This project aims to use data-driven techniques to estimate player market value and uncover hidden talent that may be undervalued in the market.

## ❓ Key Question
Can we identify undervalued football players using performance data and predicted market value?

## 🎯 Objective
- Predict player market value (`market_value_eur`)
- Analyze player performance across leagues
- Identify undervalued players based on model predictions

## 🧠 Approach (High-Level)
- Data Collection (player stats + market values)
- Data Cleaning & Preprocessing
- Feature Engineering
- SQL-based analysis
- Machine Learning model for prediction
- Power BI dashboard for scouting insights

## 🛠️ Tech Stack
- Python (pandas, numpy, scikit-learn)
- SQL (SQLite/PostgreSQL)
- Power BI
- Jupyter Notebooks
